# 📈 AI Market Dashboard

AI-powered financial analysis dashboard built with **Python + Streamlit**.  
Analyze stock market data using natural language commands with automated technical insights and visual analytics.

---

## 🚀 Features

- Natural language stock analysis
- Price trend visualization
- RSI & Moving Averages
- Correlation analysis
- Volatility metrics
- AI Buy / Sell / Hold signal
- Watchlist management
- PDF report export

---

## 🎤 Example Commands

-trend for TSLA 6 months  
-volatility of AAPL 1 year  
-RSI for NVDA 6 months  
-correlation AAPL and MSFT

---

## 🛠 Tech Stack

- **Python**
- **Streamlit**
- **Pandas / NumPy**
- **Plotly**
- **yFinance**

---

## 📊 Project Structure

